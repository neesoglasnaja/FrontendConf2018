Отброчный тур: все против всех

Sublime Text 3

  • Начать работу очень просто
  • Действительно легкий и быстрый
  • Отлично подойдет для небольших базовых проектов
  • Всего лишь блокнот
  • Нет встроенного менеджера пакетов
  • Условно-бесплатный

Atom = Sublime Text 3

Atom

  • Начать работу очень просто
  • Действительно легкий и быстрый
  • Отлично подойдет для небольших базовых проектов
  • Всего лишь блокнот
  • Нет встроенного менеджера пакетов
  • -Условно-бесплатный- Проблемы с производительностью

Visual Studio Code: выводы

  • Подойдет под проекты практически любой сложности; для некоторых технологий есть много встроенных бонусов
  • Встроенная отладка Node.js, терминал и git внутри редактора
  • Поддержка IteliSence
  • Минусы: На очень больших проектах могут быть проблемы с производительностью при поиске
  • Все еще есть баги

Web Storm

  • Может (почти) все "из коробки"
  • А что не может, есть еще доп. плагины
  • Быстрый производительный поиск по большому количеству файлов
  • Интерфейс кажется слишком сложным
  • Для простых проектов кажется overhead
  • Проблемы производительности на не очень мощных компьютерах
  • Цена
